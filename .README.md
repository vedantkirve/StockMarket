# StockMarket Trading System

A FastAPI-based algorithmic trading backend for detecting:

- Support & Resistance
- RSI support
- Fibonacci levels
- Swing-based analysis
- Custom strategy rules

## Project Structure

- `app/routes` → API endpoints
- `app/controllers` → Request handling
- `app/services` → Business logic (support, RSI, fib)
- `app/models` → Pydantic DTOs
- `app/utils` → Utility functions
- `app/core` → Config & constants

## Running the Server

```bash
source venv/bin/activate
uvicorn app.main:app --reload
```
